Web Summarizer Privacy Policy

Last updated: May 12, 2025

1. Information We Collect

• Web page text content (temporarily used for summarization only)
• User settings (API key, language preferences, font size)
• All information is stored locally on your device only.

2. How We Use Information

• Web content: Sent to OpenAI API for summarization purposes only
• Settings: Stored in browser local storage only
• No information is sent to our servers.

3. Information Sharing

• We do not share any data with third parties
• Content is only sent to OpenAI for service provision
• API keys are never transmitted externally

4. Data Storage

• All settings stored in browser local storage only
• No server-side storage
• Web content deleted immediately after summarization

5. Your Rights

• Delete stored information anytime in settings
• All data automatically deleted when extension is removed

6. Contact

For questions about this privacy policy, please contact us.

7. Changes

We will notify important changes to this policy on this page.

웹 요약기 개인정보처리방침

최종 업데이트: 2025년 5월 12일

1. 수집하는 정보

• 웹페이지 텍스트 내용 (요약 목적으로만 임시 사용)
• 사용자 설정 (API 키, 언어 설정, 글꼴 크기)
• 위 정보는 모두 사용자 기기에만 저장됩니다.

2. 정보 사용

• 웹페이지 텍스트: OpenAI API로 전송하여 요약만 수행
• 설정 정보: 브라우저 로컬 저장소에만 저장
• 어떤 정보도 저희 서버로 전송되지 않습니다.

3. 정보 공유

• 제3자와 정보를 공유하지 않습니다
• 오직 OpenAI API 서비스 제공을 위해서만 요약 대상 텍스트를 전송합니다
• API 키는 절대 외부로 전송되지 않습니다

4. 정보 저장

• 모든 설정은 브라우저 로컬 저장소에만 보관
• 서버 저장소 없음
• 웹페이지 텍스트는 요약 후 즉시 삭제

5. 사용자 권리

• 언제든지 설정에서 저장된 정보 삭제 가능
• 확장 프로그램 삭제 시 모든 정보 자동 삭제

6. 연락처

개인정보처리방침에 대한 문의사항이 있으시면 연락 주시기 바랍니다.

7. 변경사항

개인정보처리방침의 중요한 변경사항은 이 페이지를 통해 공지합니다.
